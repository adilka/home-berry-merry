# Home Berry Merry

**Home Berry Merry** — казуальная match-3 игра на Unity с яркими ягодами и уютным миром.

## Особенности

- Сотни уровней (match-3)
- Милая карта деревни
- Бустеры и награды
- Лёгкая экономика
- Кастомизация дома

## Технологии

- Unity 2022 LTS
- C# (.NET Framework)
- Git + GitHub
